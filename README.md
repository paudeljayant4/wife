## Highlights

- Animated starfield intro with a cinematic entry overlay

- Interactive 3D rose scene (click to bloom and reveal hidden message)

- Firefly ending scene and glowing pulsing heart finale
- Lightweight ambient/chime SFX generated with the Web Audio API

## Run locally
